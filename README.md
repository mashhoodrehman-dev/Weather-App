# Weather App

This is a simple weather application built using HTML, CSS, and JavaScript. It allows users to search for any city and view its current weather information using the OpenWeather API.

## Features

- Search weather by city name
- Display city and country
- Show temperature in Celsius
- Show weather description
- Display humidity
- Display wind speed
- Handle invalid city names

## Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API
- OpenWeather API

## Project Structure

```
weather-app/
│── index.html
│── style.css
│── script.js
│── README.md
```

## How to Run

1. Clone this repository.

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Open the project folder.

3. Replace the API key in `script.js` with your own OpenWeather API key.

```javascript
const api = "YOUR_API_KEY";
```

4. Open `index.html` in your browser.
